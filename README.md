# Arrows Puzzle Master Website

Bu klasör, Arrows Puzzle Master için hazırlanan resmi tanıtım, gizlilik politikası, kullanım koşulları, yardım ve veri silme sayfalarını içerir.

## Özellikler

- Arrows Puzzle Master marka ve logo kullanımı
- Türkçe / İngilizce dil seçimi
- Gizlilik Politikası
- Kullanım Koşulları
- Yardım Merkezi
- Hesap Silme ve Veri Silme talep sayfaları
- Mobil uyumlu açık tema

## Teknolojiler

- React
- React Router
- Tailwind CSS
- CRACO

## Kurulum

```bash
cd frontend
npm install
npm start
```

## Production Build

```bash
cd frontend
npm run build
```
